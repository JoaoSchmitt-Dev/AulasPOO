## 📖 Conteúdo das Aulas
 
### 🧮 Aula 01 — Calculadora
![Java](https://img.shields.io/badge/Nível-Iniciante-green?style=flat-square)
 
Primeiro contato com Java. Desenvolvimento de uma **calculadora** com operações matemáticas básicas, praticando entrada de dados, variáveis e estruturas condicionais.
 
---
 
### 🚗 Aula 03 — Introdução a Objetos
![Java](https://img.shields.io/badge/Nível-Básico-blue?style=flat-square)
 
Introdução ao paradigma de **Orientação a Objetos** com o exemplo clássico de um **Carro**. Conceitos abordados:
- Criação de Classes
- Atributos e Funções
- Métodos e comportamentos do objeto
---
 
### 🏋️ Aula 04 — Sistema de Academia
![Java](https://img.shields.io/badge/Nível-Básico-blue?style=flat-square)
 
Desenvolvimento de um **Sistema de Academia** com foco no uso de **construtores**. Conceitos abordados:
- Construtores padrão e parametrizados
- Inicialização de objetos
- Encapsulamento com getters e setters
---
 
### 👨‍🎓 Aula 05 — Sistema de Cadastro com Herança
![Java](https://img.shields.io/badge/Nível-Intermediário-orange?style=flat-square)
 
Sistema de cadastro envolvendo as classes **Pessoa**, **Aluno** e **Professor**, explorando um dos pilares da POO. Conceitos abordados:
- Herança entre classes
- Sobrescrita de métodos (`@Override`)
- Hierarquia de objetos
---
 
### 📦 Aula 09 — Sistema de Pedido
![Java](https://img.shields.io/badge/Nível-Intermediário-orange?style=flat-square)
 
Sistema de pedidos com **dependência entre objetos**, onde um `Pedido` depende de um `Cliente`. Conceitos abordados:
- Relacionamento entre classes
- Composição de objetos
- Dependência e associação
---
 
### 📚 Aula 10 — Sistema de Biblioteca
![Java](https://img.shields.io/badge/Nível-Intermediário-orange?style=flat-square)
 
**Sistema completo de Biblioteca** integrando múltiplos conceitos da POO. Conceitos abordados:
- Modelagem completa de um sistema real
- Múltiplas classes interagindo
- Aplicação prática dos pilares da POO
---
 
### 📋 Aula 11 — Objetos com Listas
![Java](https://img.shields.io/badge/Nível-Intermediário-orange?style=flat-square)
 
Manipulação de **coleções de objetos** utilizando listas em Java. Conceitos abordados:
- `ArrayList` e coleções
- Iteração sobre listas de objetos
- Gerenciamento dinâmico de dados
---
 
### 🎮 Aula 13 — Sistema de Loja de Jogos com Banco de Dados
![Java](https://img.shields.io/badge/Nível-Avançado-red?style=flat-square)
![MySQL](https://img.shields.io/badge/MySQL-4479A1?style=flat-square&logo=mysql&logoColor=white)
 
Sistema completo de uma **loja de jogos** com integração a banco de dados. Conceitos abordados:
- Conexão Java + Banco de Dados (JDBC)
- Operações CRUD
- Modelagem de dados com POO
---
 
### 🏪 Revisão Prova 1 — Sistema de Loja
![Java](https://img.shields.io/badge/Revisão-Geral-purple?style=flat-square)
 
Sistema de loja desenvolvido como revisão geral, reunindo todos os conceitos vistos. Classes implementadas:
- 👤 `Funcionario` — classe base
- 👔 `Gerente` — herda de Funcionario
- 🛒 `Vendedor` — herda de Funcionario
- 📦 `Produto` — produto comum
- 🏷️ `ProdutoPromocao` — herda de Produto com desconto
---
 
## 🚀 Como Executar
 
1. Clone o repositório
```bash
git clone https://github.com/JoaoSchmitt-Dev/AulasPOO.git
```
 
2. Abra a pasta da aula desejada no **IntelliJ IDEA**
3. Execute o arquivo `Main.java`
